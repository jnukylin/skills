# 高频模板 (Templates)

当用户需求命中下列高频场景时，直接套用模板再微调，比从零写更快更稳。每个模板都从案例库提炼而来，已经过验证。

---

## 模板 1：城市地标·毛笔浮雕风

> 适用：给某城市做艺术化地标海报，常见于文旅/文创/设计投稿

```
一支正在绘画的毛笔，描摹出一条由左下到右上的笔迹曲线，笔迹里是微观的中国[城市名]地标性景观，[主要河流/地理特征]两岸烟波浩渺，从[古建筑A]、[古建筑B]到[现代建筑C]、[现代建筑D]等地标交汇，[季节/时间特征]洒满城市的[情绪词]感，画面呈现出立体浮雕效果，毛笔笔尖在笔迹的尽头，笔迹之外都是留白，背景为纯净的白色，整体风格融合了传统中国风艺术，高细节，超写实。在画面底部笔迹开始处，手拿着毛笔在底部，有毛笔写下极小极小的瘦金体文字"[城市名]"和极小极小的英文"[CityEnglish]"，文字跟随笔迹的方向
```

**填空提示**：地标选 4-6 个，古现代各半；河流可选"运河/江水/湖面"；情绪词用"澎湃/宁静/辉煌"。

## 模板 2：城市地标·玻璃质感合集

> 适用：科技感、数字艺术风的城市名片，常见于政府/企业宣传

```
矢量插图，透明磨砂玻璃材质，[蓝/蓝白/某色]渐变，[城市名]地标建筑群合集，有立体的[地标A]、立体的[地标B]、立体的[地标C]、立体的[地标D]、立体的[地标E]等地标建筑。建筑错落有致的组合居于画面中心，建筑科技感，浅色背景，C4D，极简风格，数字艺术，背景干净纯净，极致的光线，8k超高清，正视视角视图。
```

## 使用流程

1. **识别场景**：用户需求命中哪个模板？（地标/纹样/绘本/水墨/摄影/海报/图解/手办）
2. **填空**：把 `[方括号]` 里的占位符替换成用户的具体内容
3. **微调**：根据用户额外要求加减句子
4. **质检**：过 SKILL.md 第 4 步的即梦适配度自检
5. **生成变体**：在主版基础上推一个极致版 + 一个跨界版

## 注意

- 模板是**起点不是终点**，必须根据用户需求个性化
- 填空后整体长度仍要控制在 80–250 字
- 多个模板可以混搭（如"纹样 + 玻璃质感"= 现代纹样应用）
