# 风格词典 (Style Dictionary)

用户描述笼统时（如只说"水墨""蓝色""治愈"）读本文件，把模糊词替换成**具象、可视化**的精准词汇。即梦对具象词的理解远好于抽象词。

| 金色 | 烫金 / 暖金 / 鎏金 / 浅金色勾线 |

### 配色方案（带情绪）
- **青绿山水**：石青+石绿，文人雅致
- **莫兰迪低饱和**：灰粉/灰绿/灰蓝，高级宁静
- **多巴胺撞色**：粉橙+亮黄+薄荷绿，活力治愈
- **复古胶片**：柯达Portra 800颗粒，温暖怀旧
- **唐风宫廷**：朱红+烫金+宝蓝，富丽堂皇
- **敦煌矿物**：土红+石青+蛤粉，古朴厚重
- **极简留白**：象牙白+米白+一抹主色，禅意
- **撞色实验**：靛蓝+石绿+赭石+黑白，张力十足

### 明度/饱和度描述
低饱和淡雅 / 高饱和强对比 / 漫射渐变 / 弥散晕染 / 过曝发白 / 暗调浓郁

## 使用原则

1. **替换不是堆砌**：把用户说的"好看"换成"斑驳氤氲"，不是把两个词都写上
2. **服务画面**：每个具象词都要能在脑中成像，否则不选
3. **节制使用署名**：一位艺术家署名足矣，多位会冲突
4. **配色最多 3 色主调**：超过 3 色画面会乱
