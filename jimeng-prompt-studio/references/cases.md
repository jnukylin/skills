## 类别 1：城市地标 / 微观浮雕风

**风格骨架**：一支毛笔描出曲线笔迹，笔迹内是城市地标微观景观，立体浮雕感，大面积留白，瘦金体地名+英文。

### 案例 02（苏州）
一支正在绘画的毛笔，描摹出一条由左下到右上的笔迹曲线，笔迹里是微观的中国苏州地标性景观，京杭大运河河流两岸烟波浩渺，从苏州虎丘、拙政园、狮子林、北寺塔到苏州之门、经贸大厦、金鸡湖、艺术中心等园区CBD，湖与阳光洒满城市的澎湃感，画面呈现出立体浮雕效果，毛笔笔尖在笔迹的尽头，笔迹之外都是留白，背景为纯净的白色，整体风格融合了传统中国风艺术，高细节，超写实，在画面底部笔迹开始处，手拿着毛笔在底部，有毛笔写下极小极小的瘦金体文字"苏州"和极小极小的英文"SuZhou"，文字跟随笔迹的方向

### 案例 26（济南，简化版）

### 案例 24（西安地标极简海报）
一张极简主义风格的艺术海报，核心视觉为西安地标建筑的插画，以细腻的线条和简洁的轮廓勾勒出电视塔、大雁塔、钟楼、西安奥体中心，汉城湖、等标志性建筑，背景为纯白色，营造出视觉效果，突出建筑的优雅与庄重。画面中充分借鉴詹姆斯·特瑞尔的光影幻境，以柔和的光线和渐变的色彩进行细腻的处理，使建筑在光影中若隐若现，仿佛悬浮于空中，营造出如梦似幻的仙境氛围。整体风格结合了和田诚的平面艺术简约之美，板垣光子的现代艺术手法，并融入笹尾光彦的前卫设计理念，通过将中式古典美学与现代艺术灵感相结合，提升画面的艺术张力。4K

### 案例 32（西安钟楼结构分解蓝图）
西安古城钟楼的概念分解蓝图草图，在纯白色卡片背景上放置了一个超详细的3D渲染中国古代木结构建筑组件作为中心主题，以极广角、仰视低视角的视角呈现，以产生宏大的历史深度，戏剧性的规模压缩和视觉上令人惊叹的结构张力。主要结构元件包括重檐歇山顶（青色琉璃瓦层）、复杂的斗拱互锁组件（七踩溜金斗拱）、雕刻精美的金丝楠木大柱、朱红墙体框架、藻井天花板结构、以及古城门台基，所有这些都悬挂在精确对齐的垂直轴向爆炸布局中，沿着主透视轴漂浮在空间中。主要渲染周围是密集的技术注释层：榫卯拼接序列剖面图、屋顶荷载传递路径图、藻井声学回响分析、脊兽排列顺序图和木材抗震力学标注。每个构件都标有营造尺（古代单位）和米级刻度的测量线，编号标签。

### 案例 20（铜质浮雕纹样）

### 案例 22（唐风奔马烫金纹样系统）

## 类别 3：国风水墨 / 工笔写意

**风格骨架**：大面积留白，工笔/写意，淡彩或纯墨，文人画意境。

### 案例 01（北寺塔白描）
以白描风格描绘了苏州北寺塔，蓝色调的线条具有白色背景图案，详细的插图捕捉了纯色背景前的北寺塔，这幅高清矢量手绘艺术品具有复杂的细节，其灵感来自于老式旅行书籍中的插图，采用了那个时代的插图风格

### 案例 05 / 46（街市民俗，70%留白）
大面积留白，象牙白背景，极简风格。一条曲折的街道作为画面唯一主体、两侧大面积留白。两侧留白占比至少70%，形成强烈的视觉压迫感。工笔水墨淡彩画，用极细的毫笔认真勾勒细节。呈现街市面貌，人群，拉货的马车、摊位、门店、挑担的人、挑幡算卦的、推车的人、杂耍卖艺的人与围观的人、炊烟、鸡、狗，儿童追逐。人间烟火，趣味十足。

### 案例 04（工笔莲蓬荷花）
国画工笔风格，3D效果，绿色莲蓬、粉色的荷花及贴贴层层的荷叶排列在一起，占据画面三分之二，上方大面积空白，有一个蝴蝶在飞舞，强细节，明亮画面，明亮色调

### 案例 10（蓝染工艺采蓝，剪纸版画风）
中国传统蓝染工艺第一步：采蓝，宋代院画剪纸风蓝白版画，白色背景，古代布衣女子在郊外采摘蓼蓝草，动作轻柔，衣袂飘飘，线条古朴雅致，蓝白对比鲜明。Aspect ratio: 1:1

### 案例 25（千里江山图3D卷轴）
超现实主义艺术作品，以北宋《千里江山图》为灵感，将青绿山水绘于立体卷轴之上，卷轴部分展开，呈现 3D 浮雕效果的山峦、河流、亭台楼阁，点缀骑马行人等古意人物，搭配细腻的矿物颜料质感（石青、石绿为主），背景纯白极简，光线柔和突出主体层次，融合传统工笔技法与现代 C4D 渲染，营造出"江山入画"的东方美学与当代艺术碰撞感，细节极致清晰，8K 分辨率。

### 案例 42（宋画马背书生，Bill Hocker摄影风）
Bill Hocker美学风格，黄金时刻光线捕捉，漫射光柔化肌理，水墨画的朦胧感，电影的视觉。桃花落英缤纷。宋，绿色背景，树下一匹马经过，马上坐一个书生，视顶视拍摄，真实照片，电影剧照，婆娑斑驳，光影美学，东方禅意，留白艺术，哲学隐喻，抽象思维。意境深邃，细腻肌理，柔焦模糊背景。画面带有柯达Portra 800胶片颗粒质感，非常规视角构图。高饱和强对比，极致光影，情绪表达，色彩叙事，丰富的空间细节和器物，真实照片。

## 类别 4：治愈绘本 / 日系插画

### 案例 06（槐花林治愈，新海诚风）

### 案例 09（荷叶旋转隧道，俯视涡旋）
绘本风格，色彩浓郁艳丽。梦幻主义，在一片宁静的水面上，一位个戴着斗笠的男孩，划着小船穿过由翠绿荷叶构成的旋转隧道。荷叶的纹理清晰，随着水流轻轻摇曳，形成一种梦幻般的视觉效果。整个场景被柔和而朦胧的光线所笼罩，营造出一种神秘而宁静的氛围。从下往上的视角，使得荷叶隧道更加引人注目，荷叶隧道形成旋涡视觉，引导观者的视线向中心的小船聚焦。

### 案例 27（春游野餐，Axel Scheffler+鸟山明）
以Axel Scheffler图画书风格与鸟山明手稿韵味相结合，呈现出涂鸦风格的郊外春游场景，以黄绿粉为主色调，具备线稿质感与绘本插画质感，展现大师手笔，描绘夏日野餐场景，画面中光影交错，构图巧妙，充满生机与活力的氛围。

### 案例 28（窗内治愈场景，丁达尔效应）

## 使用方法

1. **按分类定位**：根据用户需求关键词（地标/纹样/水墨/绘本/写实/3D/抽象/海报）跳到对应类别
2. **读 1-2 条**：找风格最相近的案例，提取其**句式结构**和**高级词汇**
3. **融合再创作**：把案例的骨架套到用户的具体主体上，替换关键元素，保留有效句式
4. **案例编号引用**：用户说"类似第 N 条"时，直接定位到上述案例 N
